# Peg Solitaire

Peg Solitaire is a Java project that implements the classic peg solitaire game with both a web interface and a console mode.

The web version runs as a Spring Boot application and includes game play, dummy login, scores, comments, and ratings. The project also includes REST endpoints and REST clients for working with score, comment, and rating data.

## Technologies Used

- Java
- Maven
- Spring Boot
- PostgreSQL
## Learning Purpose

I built this project to learn and practice:

- REST API
- Spring Boot application structure
- Building a small full-stack game-style application in Java
